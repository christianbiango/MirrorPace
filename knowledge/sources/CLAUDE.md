# Instructions pour Claude — Sources

- Utiliser `source_template.yaml` pour documenter toute nouvelle source.
- Ne jamais résumer une source à partir de ta propre connaissance : utiliser uniquement ce que l'utilisateur fournit (extrait, citation, lien, résumé).
- Remplir le champ `credibility` avec rigueur : `peer_reviewed` uniquement pour des études avec comité de lecture.
- Après ajout d'une source, toujours mettre à jour `../metadata/sources.yaml`.
- Ne pas remplir `extracted_principles` tant que le principe n'est pas créé dans `principles/`.
