# Research Queue

Ce dossier contient les questions auxquelles la connaissance externe doit répondre.

C'est le point de départ de tout ajout dans le Knowledge Engine.

## Utilisation

Avant de chercher une source ou d'écrire un principe :

1. Formuler la question précise à laquelle on cherche une réponse
2. L'ajouter dans `questions_to_answer.md` dans la bonne catégorie
3. Une fois une source trouvée et documentée, cocher la question et référencer la source

## Règle

Ne jamais ajouter un principe ou une règle sans qu'une question de ce fichier lui corresponde.
Les principes ne doivent pas émerger de nulle part — ils répondent à un besoin identifié.
